Suportada a versão 3.10 do Python

Para Instalar a aplicação:

```console
python3 -m venv .venv
source .venv/bin/activate
pip install pip -U pip
pip install -r requirements-dev.txt
```
Para Conferif a qualidade do Código:

```console
flake8
```
